# How to run

## javascript

- tarai.js
```bash
node javascript/tarai.js
```  

## lua

- tarai.lua
```bash
lua lua/tarai.lua
```

## python

- tarai.py

```bash
python3 python/tarai.py
```

- tarai_cache.py (cache有り)

```bash
python3 python/tarai_cache.py
```

## rust

- tarai.rs

```bash
rustc rust/tarai.rs
./tarai
```
